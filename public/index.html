'Responsive' дизайн web страницы

Адаптивный (или "отзывчивый" веб-дизайн, англ. Responsive web design) — дизайн веб-страниц, обеспечивающий отличающееся отображение сайта на различных устройствах (компьютер, планшет, смартфон). Изображение диинамически подстраивается под разные размеры окна.

Особенность подхода в том, что создается одна веб страница для всех случаев, а адаптивность достигается путем применения CSS стилей к данной странице, зависящих от размеров окна устройства

Основные принципы

в отзывчивом (responsive) дизайне:

  • применение гибкого макета на основе сетки (flexible, grid-based layout)
  • использование гибких изображений (flexible images)
  • работа с медиа-запросами (media queries)

в дополнение к этому в адаптивном (adaptive) дизайне:

  • применение постепенного улучшения (progressive enhancement)
  • проектирование для мобильных устройств с самых ранних этапов (mobile first)
